DICOM Summary

This example illustrates how to extract known elements from the data set for display in a web page. DICOM files dropped here are not uploaded anywhere - all processing is done inside your web browser in Javascript.

Usage: Drag and drop a DICOM Part 10 file into the region below for a summary of the key elements.

Note: This is a direct port of port of Cornerstone's dicomParser.js (https://github.com/cornerstonejs/dicomParser)
